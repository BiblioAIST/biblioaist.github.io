Выборы депутатов Государственной Думы Федерального Собрания Российской Федерации восьмого созыва.

Выборы депутатов Совета депутатов муниципального образования «Муниципальный округ Глазовский район Удмуртской Республики» первого созыва.

1. В 2021 году выборы продлятся три дня: 17, 18, 19 сентября. с 8:00 до 20:00 часов. Голосование на избирательных участках происходит по месту прописки, поэтому при его посещении необходимо взять паспорт. Все места для голосования можно посмотреть на интерактивной карте. А найти свой избирательный участок можно через сайт федеральной Центральной избирательной комиссии (ЦИК).

2. Проголосовать можно будет на дому. Для этого нужно заранее подать заявление в УИК, ТИК или на портале «Госуслуг». При этом у человека должна быть уважительная причина для такого способа: состояние здоровья, инвалидность, уход за больным человеком, маленьким ребёнком и иные.

3. Проголосовать можно будет через систему «Мобильный избиратель», которая позволяет избирателю голосовать на участках по месту фактического нахождения, а не по месту прописки. В этом случае гражданину вместо получения открепительного удостоверения нужно включить имя в список избирателей по месту пребывания. Сделать это можно через портал «Госуслуги», через территориальную избирательную комиссию (ТИК) и через МФЦ или обратиться в участковую избирательную комиссию (УИК). Заполнить заявление можно только лично: понадобиться только паспорт (в период замены документа - временное удостоверение личности). При этом воспользоваться механизмом «Мобильного избирателя» можно только один раз. Если гражданин предоставит несколько заявлений, то учитываться будет только самое первое.

4. По традиции на избирательных участках и избирательных комиссиях будет проводится видеосъёмка. Однако, в этом году обычные граждане не смогут получить доступ к ней. Посмотреть записи смогут члены избиркомов, политических партий, кандидаты-одномандатники и участники центров наблюдения при общественных палатах регионов.
